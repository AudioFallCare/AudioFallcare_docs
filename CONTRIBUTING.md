# Contributing Guide

AudioFallCare 프로젝트에 기여해 주셔서 감사합니다.

## 기여 방법

1. 이 레포지토리를 Fork 합니다.
2. 새 브랜치를 생성합니다 (`git checkout -b feat/new-feature`)
3. 변경사항을 커밋합니다 (`git commit -m 'feat: add new feature'`)
4. 브랜치에 Push 합니다 (`git push origin feat/new-feature`)
5. Pull Request를 생성합니다

## 커밋 메시지 규칙

| Prefix | 설명 |
|--------|------|
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 스타일 변경 |
| refactor | 코드 리팩토링 |
| test | 테스트 관련 |
| chore | 설정 변경 |

## 문의

질문이 있으시면 Issue를 생성해 주세요.
