# AudioFallCare 소프트웨어 요구사항 명세서 (SRS)

> Software Requirements Specification

---

## 1. 문서 개요

### 1.1 목적
본 문서는 AudioFallCare 시스템의 소프트웨어 요구사항을 정의한다.

### 1.2 범위
소리 기반 낙상 감지 및 보호자 알림 시스템의 기능 요구사항을 명세한다.

### 1.3 용어 정의
| 용어 | 설명 |
|------|------|
| 보호자 | 피보호자의 낙상 상태를 모니터링하는 사용자 |
| 피보호자 | 낙상 감지 대상이 되는 노약자/환자 |
| 연결 코드 | 보호자-피보호자 간 연결을 위한 일회용 코드 |
| 낙상 판별 | AI 모델을 통해 오디오에서 낙상 여부를 판정하는 과정 |
